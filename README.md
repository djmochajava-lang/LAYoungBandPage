## Development

No build process required! Simply:

1. Edit files in VS Code
2. Open `index.html` in browser
3. Refresh to see changes

## Deployment

### GitHub Pages
1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Site will be live at `https://yourusername.github.io/LAYoungArtistPage`

### Custom Domain
Point your domain (e.g., `layoung.com`) to GitHub Pages via DNS settings.
